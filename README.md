# E-commerce Customer & Funnel Analysis

This project analyzes customer behavior and sales performance using Python and Power BI to generate actionable business insights.

---

## Project Overview

The goal of this project is to:

- Understand customer purchasing behavior  
- Segment customers based on value  
- Analyze revenue contribution  
- Identify opportunities to improve conversion and retention  

---

## Dataset

The project uses:

- Customer transaction data  
- Purchase frequency data  
- Funnel stage data  

---

## Tools & Technologies

- Python (Pandas, NumPy)
- Power BI (Dashboard & Visualization)
- Jupyter Notebook
- Git & GitHub

## Key Metrics

- **Total Customers:** 49,661  
- **Total Revenue:** $681M  
- **Average Order Value (AOV):** 2.73K  

---

## Key Insights

- High-value customers contribute the majority of total revenue  
- Medium-value customers show strong potential for growth  
- Low-value customers require reactivation strategies  
- Purchase frequency strongly correlates with revenue  

---

## Business Recommendations

1. Focus retention strategies on high-value customers  
2. Increase engagement campaigns for medium-value customers  
3. Reactivate low-value customers through targeted offers  
4. Optimize pricing and bundles to increase purchase frequency
